# Учебный проект: Простой REST API на PHP

Верстка под блон в ветке master!!

Этот проект — практическая работа по созданию серверной части для блога. 
Цель: научиться работать с базой данных через PDO и обрабатывать разные типы запросов (GET, POST, PATCH).

### Как это устроено:
*functions.php* — здесь лежит вся логика: получение списка постов, чтение одного поста по ID, добавление и обновление записей[cite: 1].
*connectDb.php* — файл для подключения к базе данных `api test` через PDO.
*.htaccess** — магия для красивых ссылок. Перенаправляет запросы с любого адреса на `index.php?q=...`.

### Что умеет этот API:
1. `GET /posts` — отдает список всех постов в формате JSON.
2. `GET /posts/1` — находит и показывает один конкретный пост по его ID.
3. `POST /posts` — создает новую запись в базе данных.
4. `PATCH /posts/1` — позволяет обновить заголовок или текст уже существующего поста.
